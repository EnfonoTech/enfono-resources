DeepClaude Mixed

DeepClaude Mixed is a local proxy that lets Claude Code use DeepSeek's models for reasoning-heavy tasks while routing to Anthropic's Claude for final responses. It runs as a Node.js process on your machine and intercepts API calls — no changes needed in your Claude Code setup beyond two environment variables.

Why use it

  • Cost savings: DeepSeek is significantly cheaper than Claude for reasoning tokens
  • Flexible routing: Choose which model handles thinking vs. final output
  • Transparent: Claude Code thinks it's talking to Anthropic's API — the proxy handles the split
  • Local: Runs on your machine, no data leaves your control except to the respective APIs

Architecture

Claude Code / VS Code Extension
  │
  │  POST http://127.0.0.1:3200/v1/messages
  │  (thinks it's talking to Anthropic API)
  ▼
DeepClaude Mixed Proxy (localhost:3200)
  │
  ├─ Thinking phase → DeepSeek API (api.deepseek.com)
  │   Uses DEEPSEEK_API_KEY
  │
  ├─ Strips thinking from DeepSeek response
  │
  └─ Final response phase → Anthropic API (api.anthropic.com)
      Uses ANTHROPIC_API_KEY (or ANTHROPIC_BASE_URL)
  │
  ▼
Response returned to Claude Code (combined)

Installation

macOS

  1. Clone the repository

    git clone https://github.com/EnfonoTech/deepclaude-mixed.git
cd deepclaude-mixed

  2. Install dependencies

    npm install

  3. Set up API keys

    Add to your shell profile (~/.zshrc or ~/.bashrc):

    export DEEPSEEK_API_KEY="sk-your-deepseek-key"
export ANTHROPIC_API_KEY="sk-ant-your-anthropic-key"

    Then reload: source ~/.zshrc

  4. Start the proxy

    npm start

    The proxy starts on http://127.0.0.1:3200.

Ubuntu / Linux

  1. Install Node.js 20+

    curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs

  2. Clone and install

    git clone https://github.com/EnfonoTech/deepclaude-mixed.git
cd deepclaude-mixed
npm install

  3. Set API keys in terminal

    export DEEPSEEK_API_KEY="sk-your-deepseek-key"
export ANTHROPIC_API_KEY="sk-ant-your-anthropic-key"

    For permanent setup, add to ~/.bashrc.

  4. Start the proxy

    npm start

Windows

  1. Install Node.js 20+ from nodejs.org
  2. Clone the repo: git clone https://github.com/EnfonoTech/deepclaude-mixed.git
  3. cd deepclaude-mixed && npm install
  4. Set environment variables in System Properties → Environment Variables
  5. npm start
Auto-start: Use the install script for automatic startup on boot: 
node bin/deepclaude-mixed-setup
This creates a systemd service (Linux), launchd plist (macOS), or scheduled task (Windows).

Configuring Claude Code

Once the proxy is running on port 3200, tell Claude Code to use it:

Option A: Environment variables (recommended)

export ANTHROPIC_BASE_URL="http://127.0.0.1:3200"
export ANTHROPIC_API_KEY="unused"

Option B: Claude Code config

In ~/.claude/claude.json or VS Code settings:

{
  "api": {
    "baseUrl": "http://127.0.0.1:3200",
    "apiKey": "unused"
  }
}
Important: ANTHROPIC_API_KEY must be set to any non-empty value (e.g. "unused") when using the proxy. Claude Code requires it even though the actual API key is configured in the proxy's own ANTHROPIC_API_KEY env var.

Modes

DeepClaude Mixed supports different routing modes:

ModeThinking ModelResponse ModelBest for
mixedDeepSeek R1Claude SonnetComplex reasoning tasks (default)
deepseek-onlyDeepSeek R1DeepSeek R1Budget-sensitive, non-critical tasks
claude-only—Claude SonnetBypass proxy, use Claude directly

Switch modes via the CLI:

# Check current mode
node bin/claude-mode

# Switch mode
node bin/claude-mode deepseek-only
node bin/claude-mode claude-only
node bin/claude-mode mixed

Port already in use

If you see Error: listen EADDRINUSE: address already in use 127.0.0.1:3200:

# Find what's using port 3200
lsof -i :3200           # macOS/Linux
netstat -ano | findstr :3200  # Windows

# Kill the process
kill -9 <PID>            # macOS/Linux
taskkill /PID <PID> /F   # Windows

If the old process is a zombie (shows as (node) with no PID):

# Force kill all node processes on port 3200
fuser -k 3200/tcp       # Linux
lsof -ti :3200 | xargs kill -9  # macOS

Uninstalling

# Stop auto-start service
node bin/deepclaude-mixed-uninstall

# Remove the directory
rm -rf ~/deepclaude-mixed

# Remove env vars from ~/.zshrc or ~/.bashrc

Troubleshooting

Claude Code says "Connection refused"

The proxy isn't running. Start it: cd ~/deepclaude-mixed && npm start

DeepSeek API errors

Check your DEEPSEEK_API_KEY is valid and has credits. Test directly:

curl https://api.deepseek.com/v1/models \
  -H "Authorization: Bearer $DEEPSEEK_API_KEY"

Anthropic API errors

The proxy uses its own ANTHROPIC_API_KEY env var. Verify it's set in the terminal where you started the proxy.

VS Code: "Claude Code extension asking what's next"

This happens when the extension can't reach the API. Make sure:

  1. The proxy is running (npm start in the deepclaude-mixed directory)
  2. VS Code's Claude Code settings point to http://127.0.0.1:3200
  3. In VS Code, go to Claude Code extension settings → "Advanced: Base URL" → http://127.0.0.1:3200
Pro tip: Run the proxy in a dedicated terminal window so you can see request logs. The proxy logs every request with model routing info — useful for debugging.
