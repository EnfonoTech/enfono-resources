Weekly Project Run Checklist

CHK-ERP-003 · v1.0 · Internal Use Only. Run every week for each in-progress project. The week is one slice toward the current phase's gate — plan it Monday, execute + unblock daily, close it Friday. Companion to the Project Delivery SOP. Hit “Reset for new week” each Monday.

Owner tags: [PM] Project Manager · [FC] Functional Consultant · [TL] Technical Lead. Plan backwards from the current phase's gate; any out-of-scope ask this week → Change Request, never absorbed silently.
